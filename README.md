## Overview

Aurelia is a luxury-inspired jewelry landing page designed to present a fictional brand through a clean, elegant and responsive user interface.

The project focuses on layout structure, responsive design, visual consistency, mobile navigation behavior and basic front-end best practices.

## Technologies

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome
- Google Fonts

## Main Features

- Responsive layout built with Bootstrap
- Fixed navigation bar with mobile hamburger menu
- Hero section with background image
- Brand story section
- Product cards section
- Image carousel gallery
- Contact section with static form
- Custom JavaScript for improved mobile navigation
- Responsive behavior across desktop, tablet and mobile devices

## Sections

- Navbar
- Hero
- Brand Story
- Products
- Gallery
- Contact
- Footer

## JavaScript Behavior

The project includes a small custom JavaScript file used to improve the mobile navigation experience.

When a user selects a navbar link on mobile devices, the hamburger menu closes automatically and the page scrolls correctly to the selected section while taking the fixed navbar height into account.

## Goals

The goal of this project is to practice and showcase:

- semantic HTML structure
- responsive layout with Bootstrap
- custom CSS styling
- mobile-first design adjustments
- JavaScript behavior for mobile navigation
- visual refinement for a portfolio-ready landing page
