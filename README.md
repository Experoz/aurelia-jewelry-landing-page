# Aurelia Jewelry Landing Page

Responsive landing page for a fictional jewelry brand, built with HTML, CSS, Bootstrap and Font Awesome.

## Overview

Aurelia is a luxury-inspired jewelry landing page designed to present a fictional brand through a clean and elegant user interface.

The project focuses on layout structure, responsive design, visual consistency and basic front-end best practices.

## Technologies

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome
- Google Fonts

## Sections

- Responsive navbar
- Hero section
- Brand story section
- Product cards
- Gallery
- Contact section
- Footer

## Goals

The goal of this project is to practice and showcase:

- semantic HTML structure
- responsive layout with Bootstrap
- custom CSS styling
- navbar behavior across desktop, tablet and mobile
- visual refinement for a portfolio-ready landing page
